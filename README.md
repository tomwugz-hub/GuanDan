# 掼蛋教练 Pro 2.0

陪练 + 会解释的教练：打牌时给推荐和理由，局末告诉你最该改的几处，并记录进步。

## 快速开始

1. 安装 [Node.js](https://nodejs.org/) LTS。
2. 双击 **`点我启动掼蛋教练Pro.cmd`**（在资源管理器中打开，不要在聊天里点 `.cmd`）。
3. 浏览器会打开 **http://127.0.0.1:8010/app/**

## 一局怎么玩

| 步骤 | 操作 |
|------|------|
| 开局 | 「新开一局」单局练习，或「竞技开赛」打升级赛 |
| 出牌 | 轮到你时点「提示」看推荐；可「采纳」或自己选牌后「出牌」 |
| 提问 | 点右下角「❓ 问教练」，本机规则引擎直接作答 |
| 局末 | 右侧「本局复盘」看差异与「最该改的三处」 |
| 存档 | 点「保存复盘」；在「我的进度」→「历史复盘」翻看最近 30 局 |

不懂规则时点页头 **「规则」** 打开速查卡。

## 离线单文件版

需要不依赖本地服务时，在本目录执行：

```powershell
npm run build
```

然后打开生成的 `guandan-coach-standalone.html`（改代码后需重新 build）。

## 策略要点

1. **推荐来自规则 + 评分**，大模型只负责解释，不替你出牌。
2. **对手占牌**且有普通压牌时，优先最小代价跟牌。
3. **队友占牌**时倾向让牌，不顶队友。
4. **同花顺 / 天王炸** 非必要不拆。
5. **理牌**：连对仅当各点恰好两张，避免「假连对」。

## 开发者

旧项目 `D:\掼蛋教练` 可保留作对照；日常使用请用本目录。

### 目录结构

```
engine/      纯规则，无 UI 依赖
strategy/    理牌、评分、对手压力
coach/       回合建议、机器人、复盘
app/         浏览器界面
tests/       冒烟与关键策略回归
tools/       单文件打包、数据流水线
```

### 本地开发

```powershell
npm run dev          # 仅启动 http://127.0.0.1:8010
npm run build        # 生成 guandan-coach-standalone.html
node tests/smoke.mjs # 快速回归
```

启动脚本会顺带拉起训练采集服务（8787 端口），用于复盘反馈入库；普通打牌不依赖该服务。
