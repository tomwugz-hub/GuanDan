掼蛋教练 Pro — 快速上手

【怎么打开】
1. 安装 Node.js LTS（https://nodejs.org/）
2. 双击「点我启动掼蛋教练Pro.cmd」（或「只打开页面.cmd」）
3. 浏览器应打开：http://127.0.0.1:8010/app/?_=时间戳

请勿直接双击 html 文件（file:// 会导致页面无法加载）。
请勿用旧版「D:\掼蛋教练」文件夹里的 standalone（那是另一套程序）。

【Edge 打不开页面？】
1. 先关掉启动脚本弹出的黑窗口，再重新双击启动脚本
2. 或在黑窗口里先输入：set GUANDAN_BROWSER=chrome  再回车，然后执行：node tools\open-coach.mjs
3. 也可手动在 Chrome 地址栏打开：http://127.0.0.1:8010/app/

【改代码后页面还是旧的？】
1. 关掉所有 8010 教练标签页，再双击启动脚本（会带 ?_= 防缓存）
2. 或地址栏访问：http://127.0.0.1:8010/app/?safe=1 后点「新开一局」
3. 验证是否加载新代码：浏览器 F12 → Network → 打开 app/build.json，看 build 时间戳是否更新
4. 策略 .mjs 改完一般不用重启黑窗口；若仍异常，任务管理器结束 node.exe 后再启动

【一局怎么玩】
1. 点「新开一局」发牌（或「竞技开赛」打升级赛）
2. 轮到你时：点「提示」看推荐 → 可点「采纳」或自己选牌后「出牌」
3. 局中可随时点右下角「❓ 问教练」提问
4. 局末在右侧「本局复盘」查看差异，点「保存复盘」记录进步
5. 在「我的进度」→「历史复盘」可翻看最近保存的局

【规则不懂？】
点页头「规则」打开速查卡（牌型、进贡还贡、升级说明）。

【离线单文件版】
若需不联网使用，在项目目录执行 npm run build，再打开生成的 guandan-coach-standalone.html。

项目文件夹：D:\掼蛋教练Pro
